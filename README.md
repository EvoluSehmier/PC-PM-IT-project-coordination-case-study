# IT Project Coordination (Case Study)

## Overview
I supported the coordination of an IT project by helping teams stay aligned, organized, and on schedule.

## Objective
Ensure smooth communication between teams and help keep the project on track.

## Role
IT Project Coordinator / Business Analyst

## Tools Used
- Excel
- Jira
- Email / Communication tools

## Process
1. Gathered project requirements
2. Coordinated between teams
3. Tracked progress and tasks
4. Helped resolve issues
5. Supported project delivery

## Challenges
- Miscommunication between teams
- Delays in task completion
- Changing requirements

## Solutions
- Improved communication flow
- Kept clear task tracking
- Helped clarify requirements

## Results
- Better team alignment
- Improved project organization
- More efficient workflow
